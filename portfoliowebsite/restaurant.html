I designed and developed a full-stack restaurant management system called "Koala Restaurant" using React, JavaScript, CSS and MySQL for the fully synced front and backend. The system includes a fully functional reservation system and seamless real-time communication between front-of-house staff and the kitchen.Additionally, it has admin access to track sales, look at visualized report and edit menu and prices.

The backend provides tools for both managers and staff. Managers and staff can edit menu items and prices, and the manager has access to a dashboard for generating sales reports to analyze restaurant performance. Additionally, I implemented data visualization features to help managers interpret the data more easily.

Managers can approve or reject reservation requests through the backend, which ensures full synchronization between the front-end user interface and backend functionality. The system supports secure payment processing and is built with scalability in mind, using object-oriented design principles and patterns like Factory, Singleton, and MVC to facilitate future expansion and handle increased loads efficiently.

Functionality

The main webpage consists of an introduction to the restaurant and a button to place an order, which navigates the user to the order page. On this page, the user can see all the menu items and their prices. Users can add items to their cart, specifying the quantity for each item. The total price is dynamically updated in the cart as items are added.

Once the user has selected their items, they can proceed to the payment page. Here, they can choose between paying by cash or card. If the card option is selected, the user is prompted to enter their card details and the payment amount to proceed. For cash payments, the user enters the cash amount they are paying, and the system calculates any necessary change.

Upon successful payment, whether by card or cash, a confirmation page is displayed to the user, indicating that their order has been successfully processed. Simultaneously, the kitchen staff receives the order details in real-time, ensuring efficient communication between the front-of-house and kitchen teams.

Furthermore, the manager has access to a backend dashboard, where they can monitor all incoming orders, update the menu, view sales reports, and track reservation requests. The reservation system allows users to book tables in advance, and the manager can approve or decline reservations through the backend. This ensures that the system maintains full synchronization between user actions on the front-end and administrative functions on the back-end, creating a seamless restaurant management experience.
