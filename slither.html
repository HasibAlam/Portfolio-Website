Slither Project

The Slither Project is a decentralized Ethereum smart contract auditing platform designed to provide a seamless and secure process for detecting vulnerabilities in Solidity contracts. This project allows users to upload Solidity contracts, which are then audited using Slither, and the results are displayed on a web-based dashboard. The project integrates front-end, back-end, and API communication for an efficient auditing process.

Functionality

  • Users upload Solidity smart contracts for auditing.
  • The backend processes the contract and runs Slither for analysis.
  • The audit results are stored in a MySQL database.
  • The FastAPI API fetches audit results and delivers them to the frontend.
  • Users can view audit reports, download them, and see audit history.
  • The dashboard provides a real-time view of vulnerabilities detected.

Backend, Frontend, and API

The backend is built using Python with FastAPI for the API services. It processes the contracts, triggers the Slither analysis, and stores results in MySQL. The frontend is developed with React, which provides a responsive and user-friendly interface for interacting with the auditing system and viewing results. FastAPI ensures fast, secure, and scalable communication between the front-end and back-end systems.
